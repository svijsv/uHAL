typedef long int adcm_t

Type used to perform mathematical functions with ADC readings.

This is needed for operations which may overflow adc_t such as multiplication or those which may involve negative values.

Note

adc_t adc_read_ac_amplitude ( gpio_pin_t  pin,
uint_fast32_t  period_ms,
adc_t *  min,

)

Try to find the amplitude of an AC voltage.

Attention
This function does not respect ADC_SAMPLE_COUNT.
Parameters
pinThe pin to examine.
period_msThe time to spend monitoring in milliseconds.
minIf non-NULL, set to the lowest ADC reading encountered.
maxIf non-NULL, set to the highest ADC reading encountered.
Returns
Half the difference between the high and low peaks, or ERR_ADC if there's an error.
