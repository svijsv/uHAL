Enable hardware PWM output on an IO pin.

Attention
The pin mode may be changed by the backend when PWM is enabled.
Parameters
outputThe handle used to manage the pin. Must not be NULL.
pinThe pin to enable PWM on. If 0, attempt to use the last pin associated with the handle.
duty_cycleThe initial duty cycle of output.
Returns
ERR_OK if successful, otherwise an error code indicating the nature of the problem encountered.
