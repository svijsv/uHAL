Detailed Description

SD card SPI driver derived from an example found at http://elm-chan.org/fsw/ff/00index_e.html.

Attention
This driver requires the FatFS headers be available in the C header search path.
Because this driver is intended for use by FatFS, the return codes differ from those used throughout the rest of the HAL.
The macro SPI_CS_SD_PIN must be set to the gpio_pin_t of the SD card's chip-select pin.
